What's New

Version 0.3 - Organization & Export ▼
  • 🔍 Filter & Search: Filter events by type (Community Day, Raids, etc.) and search by name
  • ⭐ Favorites System: Star important events to find them quickly - favorites persist between app launches
  • 📆 Calendar Export (macOS): Add events directly to your macOS calendar with automatic timezone conversion
  • 💾 Smart Storage: Favorites saved locally with SwiftData for privacy and speed
  • 🎯 Filter Badge: See how many filters are active at a glance
Version 0.2 - Pokemon Details & Images ▼
  • ✨ Event Images with thumbnails in lists and hero images in details
  • 👾 Pokemon Details for featured Pokemon, raid bosses, and Community Day Pokemon
  • 🌟 Shiny Badges showing which Pokemon are available as shinies
  • ⏱️ Live Countdowns with progress bars for active events
  • 🎨 Color-Coded Types for quick identification
