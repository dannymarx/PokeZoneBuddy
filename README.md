# PokeZoneBuddy

**PokeZoneBuddy** ist eine macOS- und iOS-App, die Pokémon GO-Events anzeigt und automatisch die lokalen Zeiten für deine Lieblingsstädte umrechnet.  
So weißt du immer genau, wann du zu Hause aktiv sein musst, um ein Event mitspielen zu können, das z. B. in Tokio oder New York stattfindet.

---

## ✨ Features
- Anzeige kommender Pokémon GO Events
- Favoritenliste mit eigenen Städten
- Automatische Umrechnung in deine lokale Zeit


➡ [Siehe Changelog](./CHANGELOG.md) für Details zu allen Versionen.

## 📊 Datenquelle

Diese App verwendet Event-Daten von:
- **[ScrapedDuck](https://github.com/bigfoott/ScrapedDuck)** (MIT License) - API für Pokémon GO Event-Daten
- **[LeekDuck.com](https://leekduck.com)** - Original-Datenquelle (mit Erlaubnis gescraped)
